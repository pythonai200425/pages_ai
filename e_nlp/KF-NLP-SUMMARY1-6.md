# spaCy Cheat Sheet Summary

## מה חשוב באמת לדעת על spaCy

> `spaCy` היא ספריית NLP מהירה וחזקה לניתוח שפה טבעית, עם דגש על עבודה מעשית עם טקסט אמיתי

## היכולות המרכזיות

| נושא | מה זה אומר בקצרה |
|---|---|
| Tokenization | פירוק טקסט למילים, סימנים וקיצורים בצורה חכמה |
| POS Tagging | זיהוי סוג המילה: פועל, שם עצם, תואר ועוד |
| Dependency Parsing | הבנת המבנה התחבירי: נושא, פועל, מושא וקשרים בין מילים |
| NER | זיהוי ישויות בשם כמו אנשים, מקומות, ארגונים, תאריכים וכסף |
| Lemmatization | החזרת מילה לצורת הבסיס שלה |
| Sentence Segmentation | חלוקת טקסט למשפטים |
| Noun Chunks | זיהוי צירופים שמניים חשובים |
| Stop Words | זיהוי מילים נפוצות שפחות חשובות לתוכן |
| PhraseMatcher | זיהוי ביטויים קבועים מתוך רשימה |
| displaCy | הצגה ויזואלית של תחביר וישויות |

## שלושת האובייקטים שצריך לזכור

| אובייקט | תפקיד |
|---|---|
| `Doc` | כל הטקסט אחרי עיבוד |
| `Token` | יחידה אחת מתוך הטקסט |
| `Span` | רצף טוקנים מתוך `Doc` |

## תכונות חשובות של Token

- `token.text` = הטקסט עצמו
- `token.pos_` = סוג המילה
- `token.dep_` = התפקיד התחבירי
- `token.is_stop` = האם זו מילת עצירה
- `token.is_sent_start` = האם זו תחילת משפט
- `token.lemma_` = צורת הבסיס

## דוגמת עבודה בסיסית

```python
import spacy

nlp = spacy.load("en_core_web_sm")
doc = nlp("Apple is opening a new office in London for $5 million")
```

מה מקבלים מהשורה הזו:

- טוקנים
- תגיות דקדוקיות
- מבנה תחבירי
- ישויות בשם
- חלוקה למשפטים

## ישויות נפוצות שצריך להכיר

| תווית | פירוש |
|---|---|
| `PERSON` | אדם |
| `ORG` | ארגון או חברה |
| `GPE` | מדינה, עיר או אזור |
| `DATE` | תאריך |
| `MONEY` | סכום כסף |

## למה הטוקניזציה של spaCy נחשבת טובה

- לא מפרקת רק לפי רווחים
- יודעת לטפל בקונטרקציות כמו `we're`
- מזהה קיצורים כמו `U.S.` או `L.A.`
- מפרידה סימני פיסוק בצורה חכמה

## Lemmatization מול Stemming

| נושא | Lemmatization | Stemming |
|---|---|---|
| דיוק | גבוה יותר | נמוך יותר |
| תוצאה | מילה תקינה | שורש גס |
| מבוסס הקשר | כן | לא |
| ב-spaCy | קיים | לא מובנה |

> כלל מהיר: אם חשוב להבין משמעות בצורה נכונה, עדיף `lemmatization`

## כלים שימושיים במיוחד

### `doc.ents`
מחזיר את כל הישויות בשם שזוהו בטקסט

### `doc.sents`
מחזיר את המשפטים בטקסט

### `doc.noun_chunks`
מחזיר צירופים שמניים חשובים

### `spacy.explain(...)`
מסביר תגיות כמו `ORG`, `AUX`, `nsubj`

### `PhraseMatcher`
מזהה ביטויים מתוך תבניות מוכנות מראש

### `displacy.render(...)`
מציג בצורה גרפית תחביר או ישויות

## מתי משתמשים במה

| צורך | כלי |
|---|---|
| פירוק טקסט למרכיבים | Tokenization |
| הבנת סוגי מילים | POS |
| הבנת מבנה המשפט | Dependency Parsing |
| זיהוי אנשים, מקומות, ארגונים | NER |
| ניקוי רעש בטקסט | Stop Words |
| זיהוי ביטויים קבועים | PhraseMatcher |
| הצגה ויזואלית | displaCy |

## שורת סיכום לבחינה

> `spaCy` היא ספריית NLP שמבצעת עיבוד טקסט מלא: טוקניזציה, זיהוי חלקי דיבר, ניתוח תחבירי, זיהוי ישויות, למיטיזציה, חלוקה למשפטים וחילוץ ביטויים חשובים

## לזכור ב-10 שניות

- `Doc` = כל המסמך
- `Token` = מילה או סימן
- `Span` = קטע מתוך המסמך
- `NER` = זיהוי ישויות בשם
- `lemma` = צורת בסיס של מילה
- `displaCy` = ויזואליזציה
- `PhraseMatcher` = זיהוי ביטויים
- `spaCy` טובה ליישומים מעשיים, מהירים ומדויקים
